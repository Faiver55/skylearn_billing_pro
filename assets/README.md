# Skylearn Billing Pro - Assets

This directory contains static assets for the plugin.

## Directory Structure

- `/css/` - Stylesheets
- `/js/` - JavaScript files  
- `/images/` - Images, icons, and logos
- `/fonts/` - Custom font files (if needed)

## Usage

Assets are enqueued properly through WordPress hooks in the main plugin file.

## Branding Assets

Logo and icon files should be placed in the `/images/` directory following the style guide specifications.
